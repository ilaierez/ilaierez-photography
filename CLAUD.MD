# Project: Sports Photography Website

## Overview
This project is a clean, modern website for a sports photography business.  
The goal is to showcase high-quality sports photos, attract clients, and allow easy contact.

## Pages Structure
The website should include 3 main pages:

1. Home Page
2. Portfolio Page
3. Contact Page

---

## Design Guidelines

- Style: Modern, clean, sporty
- Colors: Dark background (black / dark gray) with white text and accent color (red / neon / blue)
- Typography: Bold headlines, simple readable body text
- Layout: Responsive (mobile + desktop)

---

## Home Page

### Sections:
- Hero section with a strong sports image
- Title: "Sports Photography" or business name
- Short tagline (e.g. "Capturing the action")
- Call-to-action button: "View Portfolio"
- Small intro about the photographer

---

## Portfolio Page

### Requirements:
- Grid gallery layout
- Images displayed in rows (3–4 per row on desktop)
- Click on image → opens larger view (lightbox)
- Smooth hover effects
- Only selected images should be displayed

---

## Contact Page

### Include:
- Contact form:
  - Name
  - Email
  - Message
- Submit button
- Optional:
  - Phone number
  - Instagram link
- Simple success message after sending

---

## Functionality

- Navigation menu between pages
- Smooth scrolling / transitions
- Responsive design
- Fast loading images (optimized)

---

## Tech Stack (recommended)

- Frontend: HTML, CSS, JavaScript
- Optional: React or Next.js
- Styling: Tailwind CSS (preferred)

---

## File Structure (suggested)

- /index.html (Home)
- /portfolio.html
- /contact.html
- /css/
- /js/
- /images/

---

## Extra Features (optional)

- Instagram feed integration
- Lazy loading images
- SEO optimization
- Image categories (football, basketball, etc.)

---

## Tone

Professional, energetic, and sports-focused.
The site should feel dynamic and action-driven.
